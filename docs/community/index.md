## Ways to Contribute

- **Code**: Fix bugs, implement features, improve performance
- **Documentation**: Improve docs, write tutorials, fix errors
- **Design**: Create better logos, themes, wallpapers
- **Testing**: Test on different hardware, report issues
- **Plugins**: Build integrations and extensions
- **Localization**: Translate the OS and documentation
- **Community**: Help others, answer questions, spread the word

## Development Discussions

- **Weekly Dev Calls**: Every Wednesday at 18:00 UTC
- **Architecture Reviews**: Monthly deep-dive sessions
- **Plugin Showcase**: Monthly demo of community plugins

# User Guide

This guide covers everything you need to use Prometheus OS effectively.

## Quick Reference

| Task | How |
|------|-----|
| Open AI prompt | ++super+space++ |
| Search for files | ++super+f++ |
| Switch workspace | ++super+1++ to ++super+9++ |
| Open terminal | Say "open terminal" or press ++super+enter++ |
| Take screenshot | ++super+shift+s++ |
| System dashboard | Run `prometheus-dashboard` |
| Settings | Say "open settings" or run `prometheus-settings` |

## The AI Assistant

The AI is your primary interface to the OS:

```bash
prometheus-ai --query "Open my project files"
prometheus-ai --query "What's using my CPU?"
prometheus-ai --interactive   # Start a conversation
```

## Desktop

Two desktop environments are available:

1. **Prometheus Native** — Custom compositor, maximum performance
2. **Prometheus GNOME** — Full GNOME desktop with AI integration

Select at the login screen.
